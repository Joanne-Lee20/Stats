Real-World Applications of Machine Learning
SAROPS|

...

  • > Stands for Search and Rescue Optimal Planning System; refers to the software the U.S. Coast Guard utilizes for maritime searches
  • > A Monte Carlo based system; uses simulated particles generated by user inputs in a wizard based Graphical User Interface (GUI)
  • > Capable of handling multiple scenarios and search object types, model pre-distress motion and hazards, and account for the affects of previous searches
  • > Makes requests to and receives from an Environmental Data Server (EDS), which provides both real-time and historical data about the environment
  • > Also allows manual inputs of winds and currents input via a "sketch" tool using objective analysis techniques
  • > Projects the drift of the survivors and craft using advanced drift algorithms
  • > Automates Search Rescue Unit (SRU) allocation by maximizing Probability of Success (POS); each SRU gets a recommended search pattern that accounts for the relative motion between the SRU and the drifting particles (achieved by using the Probability of Detection as function Lateral Range to update the probability of detection for each particle)


Speech Recognizers|

...

  • > Speech recognition may also be called automatic speech recognition (ASR), computer speech recognition, or speech-to-text; recognizers utilize natural language processing (NLP) to detect and read human speech, converting it to a written format
  • > Considered to be one of the most complex areas of computer science, as human language can be difficult to process accurately
  • > Often incorporated into the systems of mobile devices to facilitate communication through texting or to access virtual agents such as Google Assistant or Apple's Siri to complete tasks such as voice search; Amazon's Alexa or Microsoft's Cortana can be used to play music
  • > Made up of a few components: speech input, feature extraction, feature vectors, decoder, word output
  • > Evaluated on accuracy rate, such as speed and word error rate (WER), which can be impacted by factors such as pronunciation, accent, pitch, volume, and background noise
  • > Improves driver safety by enabling voice-activated navigation systems and search capabilities in car radios
  • > Used by those in the medical field to capture and log patient diagnoses and treatment notes


Customer Service Technology|

...

  • > Used to answer frequently asked questions (FAQs) and provide personalized advice about products for users
  • > Includes messaging bots on online sites with virtual agents, messaging apps, and tasks usually done by virtual assistants and voice assistants
  • > Removes the need to wait for a human employee to be available; incorporation of speech recognition systems help reduce time to resolution for consumer issues


Recommendation systems/engines|

...

  • > Generally concerned with ranking or rating products and users; refers to a system that predicts the rating a user might give to a specific item, which are then also ranked or rated and returned back to the user
  • > AI algorithms can help discover data trends using past consumer behavior data to develop more effective cross-selling strategies; typically used to make relevant recommendations to customers during online checkout process
  • > Often used by companies such as Google, Instagram, Spotify, Amazon, Tiktok and Netflix to increase engagement (For instance, Spotify recommends songs similar to ones the user has repeatedly listened to in order to encourage continued use of the platform)
